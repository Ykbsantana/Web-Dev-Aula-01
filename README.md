# Entrega I - Plataforma ONG (Fundamentos e Estruturação)


Instruções rápidas:


1. Mantenha todos os arquivos em uma pasta pública do GitHub (repositório público).
2. Otimize imagens antes de subir (ex.: tools como ImageOptim ou sites de compressão) e gere múltiplos formatos se possível (.webp, .jpg).
3. Valide os HTML no W3C Validator e corrija avisos relevantes.
4. Teste responsividade em diferentes breakpoints e a navegação por teclado.


Boas práticas recomendadas para a correção:
- Semântica HTML correta (usar header, main, nav, footer, article, section, figure, figcaption, address, form, fieldset, legend).
- Formulários com atributos required, pattern, type corretos.
- Scripts com comportamento não intrusivo (defer) e sem dependência externa.
